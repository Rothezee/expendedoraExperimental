# Plan activo

El ciclo **plan-spec → plan-build → plan-review** usa un único archivo:

- **`active-plan.md`** — plan en curso (creado por `plan-spec`, actualizado por build/review)

Plantilla: `.cursor/skills/plan-spec/plan-template.md`

Los planes archivados pueden guardarse aquí como `YYYY-MM-DD-slug.md` al cerrar un ciclo.
